# Enabler  [![CircleCI](https://circleci.com/gh/nicolastrres/enabler.svg?style=svg)](https://circleci.com/gh/nicolastrres/enabler)

# Running service locally
```
docker-compose up
```


# Running tests
```
yarn install
yarn lint
yarn test
````
